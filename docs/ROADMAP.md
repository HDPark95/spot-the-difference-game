# 틀린그림찾기 게임 로드맵

## 현재 상태
- React + Vite, HashRouter
- 8레벨 (cafe, livingroom, park, dogwalk, coffee, cooking, classroom, lesson)
- Storyset SVG 기반, Puppeteer DOM 조작으로 차이점 생성
- 픽셀 검증 자동화 (히트맵 PNG, A/B/C/F 등급)
- GitHub: https://github.com/HDPark95/spot-the-difference-game

---

## Phase 1: 게임 품질 개선

### 1.1 C등급 차이점 전부 B등급으로 개선
- cafe #2 선반 병 (4.9%) → 더 넓게 제거
- cafe #5 카운터 장식 (9.0%) → 더 큰 요소로 교체
- livingroom #2 선반 받침대 (6.2%) → 대상 확대
- park #3 나비 (3.7%) → 더 큰 나비 or 다른 변경
- lesson #1 칠판 내용 (6.8%) → 범위 확대

### 1.2 차이점 유형 다양화
- 현재: 제거/색변경/이동/반전
- 추가: 오브젝트 교체 (컵→찻잔), 패턴 변경, 크기 미세 조절
- 컴포넌트 라이브러리(87개) 적극 활용

### 1.3 레벨 추가 (20개 목표)
- Storyset 장면 추가 다운로드 (부엌, 사무실, 해변, 캠핑 등)
- 게임 크리에이터 스킬로 자동 생성 → 픽셀 검증 → 배포
- `.remove()` 방식 통일 (display:none 안 먹히는 SVG 있음)

### 1.4 숨은그림찾기 모드
- 같은 SVG 인프라 활용
- 장면 1개 + 하단에 "찾아야 할 물건" 리스트
- 컴포넌트 라이브러리에서 소품을 장면에 교묘하게 삽입
- 클릭하면 찾은 것으로 처리

## Phase 2: UX/UI 개선

### 2.1 점수 시스템
- 시간 기반 점수 (빠를수록 높은 점수)
- 힌트 사용 시 감점
- 오답 클릭 시 감점
- 레벨별 별 등급 (3성/2성/1성)

### 2.2 진행 저장
- localStorage로 클리어 기록 저장
- 홈 화면에서 클리어한 레벨 표시 (체크마크, 최고 기록)
- 잠긴 레벨 (이전 레벨 클리어 시 해금)

### 2.3 효과음/진동
- 정답 시 효과음 + 모바일 진동
- 오답 시 효과음
- 클리어 시 축하 효과음
- Web Audio API 사용

### 2.4 줌 기능
- 핀치 줌 (모바일)
- 스크롤 줌 (데스크톱)
- 작은 차이점 찾기에 필수
- react-zoom-pan-pinch 라이브러리

### 2.5 테마
- 다크 모드 (현재) / 라이트 모드 토글
- 색상 팔레트 커스텀

## Phase 3: 기술 고도화

### 3.1 배포 자동화
- GitHub Pages 또는 Vercel 자동 배포
- CI/CD: push 시 자동 빌드 + 픽셀 검증 + 배포
- GitHub Actions 워크플로우

### 3.2 PWA
- Service Worker로 오프라인 플레이
- 홈 화면에 추가 가능
- SVG 에셋 캐싱

### 3.3 레벨 자동 생성 파이프라인
- SVG URL 입력 → 분석 → 차이점 자동 설계 → 변형 → 픽셀 검증 → levels.js 추가 → 빌드 → 배포
- 전 과정 자동화 (사람 개입 없이)
- 게임 크리에이터 스킬 + 픽셀 검증 스킬 연동

### 3.4 검증 강화
- pixel-verify.mjs 개선: C등급 이하 자동 리젝트
- 히트박스 겹침 자동 감지 + 경고
- 히트박스 밖 변경 자동 보정
- 게임 플레이 테스트 (Puppeteer로 전 레벨 자동 클리어)

## Phase 4: 비즈니스 확장

### 4.1 일일 챌린지
- 매일 새 문제 자동 생성 (Storyset에서 랜덤 SVG → 자동 파이프라인)

- 연속 출석 보상

### 4.2 멀티플레이어

- 사용자가 SVG 업로드
- 자동 분석 → 차이점 후보 제안
- 사용자가 차이점 선택/수정
- 공유 링크 생성

### 4.4 통계/랭킹
- 전체 플레이어 랭킹
- 레벨별 평균 클리어 시간
- 개인 통계 (총 플레이 시간, 정확도 등)
